## Turni Smart — GTT

Web app per la gestione dei turni GTT (Gruppo Torinese Trasporti).

## Funzionalità

- **Preconoscenza**: carica il PDF mensile e consulta i turni per data
- **Orari Linee**: carica il PDF del deposito per lo sviluppo completo dei turni
- **Calendario mensile**: vista a griglia con tap per dettaglio
- **Navigazione per linea**: esplora turni per numero di linea e tipo giorno
- **Ballottaggio**: incolla il turno assegnato direttamente nella card
- **Condivisione**: Google Calendar, Calendario iOS, WhatsApp, Copia
- **Moovit**: percorso bus verso il luogo di inizio turno
- **Storico**: i documenti caricati restano memorizzati nel browser

## Utilizzo

Apri `index.html` nel browser e carica i PDF dei turni.
